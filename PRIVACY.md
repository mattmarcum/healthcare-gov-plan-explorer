# Privacy Policy

_Last updated: 2026-06-25_

**Plan Explorer for HealthCare.gov** ("the extension") is an independent browser
extension that helps you view and compare Marketplace health plans on HealthCare.gov.

## The short version

The extension collects **no data**. It has no backend server, no analytics, no
advertising, and no account. Nothing you do in the extension is transmitted to
the developer or any third party.

## What the extension accesses

When you click **Compare all plans** on your HealthCare.gov enrollment page, the
extension, running locally in your browser:

1. Reads the search parameters for **your own** application (household members,
   ZIP/county, subsidy amount, and cost-sharing level) from HealthCare.gov,
   using the session you are already logged into — the same data the
   HealthCare.gov website itself uses to show you plans.
2. Sends those parameters to HealthCare.gov's own public plan-search API to
   retrieve the list of plans available to you.
3. Displays the results in a table in your browser.

All of this happens between your browser and HealthCare.gov. The developer of
this extension never receives any of it.

## What the extension does **not** do

- It does not send your data to the developer or any third-party server.
- It does not use analytics, tracking, cookies of its own, or advertising.
- It does not store your personal or health information. (Optionally, you can
  export the on-screen plan table to a CSV file saved to your own computer; that
  file is created locally and is not uploaded anywhere.)
- It does not run on any website other than HealthCare.gov enrollment pages.

## Permissions

The extension requests host access to `www.healthcare.gov` and the HealthCare.gov
marketplace plan API solely to perform the function above. It requests no other
permissions.

## Affiliation

This extension is an independent project. It is **not** affiliated with,
endorsed by, or operated by HealthCare.gov, the Centers for Medicare & Medicaid
Services (CMS), or the U.S. government.
